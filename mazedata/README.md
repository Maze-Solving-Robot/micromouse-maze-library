# 迷路データ

マイクロマウスの迷路データ

## フォーマット

ファイルはテキスト形式．以下のパーツを用いて迷路情報を表現する．

| パーツ       | 表現形式 | 注記                   |
| ------------ | -------- | ---------------------- |
| 柱           | `+`      | 隣接壁がないときも記載 |
| 既知壁（横） | `---`    |                        |
| 既知壁（縦） | `|`      |                        |
| 未知壁（横） | ` . `    | 必要に応じて使用する   |
| 未知壁（縦） | `.`      |                        |
| スタート区画 | ` S `    | 1区画のみ              |
| ゴール区画   | ` G `    | 複数区画可             |
| その他の区画 | `   `    |                        |

注意:

- 迷路サイズは正方形に限る．長方形の迷路を表現したい場合は長辺に合わせ，左下に寄せる．
- ファイルの終端に改行をつけること

### 例

```txt
+---+---+---+---+---+---+---+---+---+
|   |       |       |       | G   G |
+   +   +   +   +   +   +   +   +   +
|       |       |       |   | G   G |
+   +   +---+   +   +---+   +---+   +
|   |           |       |           |
+   +---+   +---+   +---+---+   +---+
|       |       |           |       |
+---+   +   +   +---+---+   +---+   +
|       |   |       |       |       |
+   +   +---+---+   +   +   +   +   +
|   |   |       |       |       |   |
+   +   +   +   +---+   +---+   +   +
|   |       |       |           |   |
+   +---+   +---+   +---+   +---+   +
|       |   |       |           |   |
+   +   +   +   +---+   +---+   +   +
| S |       |                       |
+---+---+---+---+---+---+---+---+---+
```
