Detailed Description

Position と Direction をまとめた型．位置姿勢．

位置姿勢は，区画とそこに向かう方向で特定する． 現在区画から出る方向ではないことに注意する．

+---+---+---+ 例:
| < | <--- (0, 2, West)
+ +---+ ^ + <--- (2, 2, North)
| > | <--- (1, 1, East)
+ +---+ v + <--- (2, 0, South)
| S | | <--- (0, 0)
+---+---+---+
